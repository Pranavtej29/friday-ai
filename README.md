# 🤖 Friday AI Assistant

A voice-powered AI assistant inspired by Friday from Iron Man.
Talk to it and it talks back!

## Features
- 🎤 Hears your voice
- 🧠 Thinks using AI
- 🔊 Talks back to you

## How to run it

### Step 1 - Install Python
Download from python.org

### Step 2 - Install libraries
pip install SpeechRecognition pyttsx3 groq sounddevice numpy

### Step 3 - Get a free Groq API key
Go to console.groq.com and sign up for free

### Step 4 - Add your key
Open friday.py and replace YOUR_KEY_HERE with your actual key

### Step 5 - Run it
python friday.py

### Step 6 - Talk to Friday!
Just speak into your mic and Friday will reply!
